# ADR 0003 — Provenance is structural, not conventional
**Status** accepted (Phase 1, extended Phase 2) · **Decision** every
numeric carries a ref (packet/store/derived/prior/ledger). A verifier
RESOLVES refs server-side; one failure rejects the whole card. Analyst
prose passes a numeric citation gate against its evidence pack
(lower-precision quoting ok; invented precision fails).
**Why** "no UI number without a source" only survives if violations are
mechanically impossible to ship. The Analyst inherits the gate unchanged.
