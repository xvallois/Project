# ADR 0007 — Analyst trust model
**Status** locked by product owner (pre-Phase 2) · **Decision** the
Analyst never creates opportunities/data/analytics and never overrides
bands. Input = numbered evidence pack only. Output = strict 7-section
ResearchBrief, statements typed analyst|deterministic, citation-gated.
Visible separation in UI (ENGINE amber / ANALYST violet, provider badge).
Budget: 1/3/10u tiers, hard daily 100u, 20% carryover, 12u triage
reserve; zero budget never degrades deterministic workflows.
**Why** the workstation is a trading platform with an Analyst attached —
never a chat app with trading widgets.
